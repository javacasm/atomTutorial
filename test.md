# Keybinding

* Ctrl+Alt+m  Markdown pre-visualización
* Ctrl+ + y Ctrl + - Aumenta y disminuye el tamaño el font
### Git

* Ctrl-Alt-a Add file
* Ctrl-Alt-x Commit
